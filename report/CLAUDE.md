# 项目报告要求
- 标题层级不超过三层，有序/无序列表层级不超过两层
- 严格按照中文论文的行文风格协作。行文必须使用自然语言表示上下文逻辑，非必要情况（即非系统展示部分）下严禁使用三、四级标题或有序、无序列表等硬分点的方式表示枚举。
- 学术写作规范，提升报告的品质与影响力，精细润色每个细节，优化语言表达与逻辑结构。
- 除正文外还需要包含论文应该有的其他内容，如标题、摘要、目录、引用。
- 使用 typst 进行写作，语言为中文。尽可能将文章模块化，创建多个 .typ 进行写作。
- 可以引用参考文献，包含公式和图片，图片使用占位符
- 可以上网搜索额外的相关资料
- 仔细思考，全面规划后再写作。
